## 0. Context

AOWATT Global Materials 化工 B2B 官网。现状：
- 文档完备（`CLAUDE.md` / `docs/ARCHITECTURE.md` / `docs/TECH_STACK_DECISION.md` / `docs/architecture-review-and-roadmap.md` 已对齐 V0.1/V0.2/V0.3 路线图与所有关键决策）。
- Figma 设计稿 7 个页面（Home / Products / Product Detail / Why Us / About / Applications / Contact）骨架完整，Header/Footer/PageHero/卡片宽度跨页统一。所有 visual 都是矩形占位，没有真实图片。
- 代码骨架尚未初始化。`src/` 不存在。

V0.1 不做：询盘表单、API Route、邮件服务、Turnstile、Supabase、后台、AI 客服、文档下载（COA/TDS/SDS）。

V0.1 必须满足 ARCHITECTURE.md 第 17 节 Go/No-Go 上线门禁。

用户已确认的关键决策：
- 占位视觉风格：几何 SVG 抽象图（5 条粗细不一水平条 + gold ring icon + 品牌色背景），跨页统一，无需真实图片。

- 法律页文案：本 spec 用通用模板生成 Privacy Policy 与化工 Disclaimer 草稿，关键字段标 `[PLACEHOLDER]`，上线前由业务方/律师确认。

## 1. 架构总览

```
┌──────────────────────────────────────────────────────────┐
│              Static Next.js (App Router)                  │
│  app/[locale]/{home, products, contact, privacy, …}      │
│           ↓ uses                                          │
│  components/{layout, ui, home, products, ...}            │
│           ↓ reads via                                     │
│  lib/{products, categories, i18n, seo, analytics}        │
│           ↓ from                                          │
│  content/{site, categories, products}                    │
│  config/features.ts (page on/off)                        │
│  messages/{en,zh}.json (next-intl)                       │
└──────────────────────────────────────────────────────────┘
        ↓ pnpm build (output: 'export')
   Static HTML + JS → Vercel Static Deployment
        ↓
   Cloudflare DNS / CDN / Web Analytics (passive)
   Vercel Web Analytics (custom events)
   Google Search Console (search performance)
   Sentry (errors)
```

页面只展示，不直接 import 产品数据；产品/分类通过数据访问层读取，未来切到 Supabase 时只改 `lib/`，不动 `app/` 或 `components/`。

### 5.6 数据访问层（`src/lib/products.ts` / `categories.ts` / `i18n.ts`）

`lib/products.ts` 暴露：
```ts
getPublishedProducts(locale: Locale): LocalizedProduct[]
getFeaturedProducts(locale: Locale, limit?: number): LocalizedProduct[]
getProductBySlug(slug: string, locale: Locale): LocalizedProduct | null
getProductsByCategory(categorySlug: string, locale: Locale): LocalizedProduct[]
getRelatedProducts(product: LocalizedProduct, locale: Locale, limit?: number): LocalizedProduct[]
getProductStaticParams(): { locale: Locale; slug: string }[]
searchProducts(query: string, locale: Locale): LocalizedProduct[]
```

`LocalizedProduct` 是 `Product` 经过 `lib/i18n.ts` 的 `localize()` 处理后的版本：每个 `LocalizedString` 字段被解包成 `string`（`zh` 缺失时 fallback `en`）。

`lib/i18n.ts`：
```ts
export function pickLocalized<T>(s: { en: T; zh?: T }, locale: Locale): T {
  return (locale === 'zh' && s.zh !== undefined) ? s.zh : s.en;
}
export function hasZhFallback(s: { en: unknown; zh?: unknown }, locale: Locale): boolean {
  return locale === 'zh' && s.zh === undefined;
}
```

页面在中文场景检测到任何 `hasZhFallback` 命中时显示一次顶部 banner：「中文版正在更新，部分内容暂以英文显示」。

## 8. SEO / Sitemap / Robots / Error Pages

### 8.1 `lib/seo.ts`

```ts
type BuildArgs = {
  title: string; description: string; path: string; locale: Locale;
  ogImage?: string;
};

export function buildMetadata({ title, description, path, locale, ogImage }: BuildArgs): Metadata {
  const base = process.env.NEXT_PUBLIC_SITE_URL ?? 'https://aowatt.com.au';
  const url = `${base}/${locale}${path}`;
  return {
    title, description,
    alternates: {
      canonical: url,
      languages: {
        'en': `${base}/en${path}`,
        'zh': `${base}/zh${path}`,
        'x-default': `${base}/en${path}`,
      },
    },
    openGraph: {
      title, description, url, siteName: 'AOWATT Global Materials',
      locale: locale === 'zh' ? 'zh_CN' : 'en_AU',
      type: 'website',
      images: ogImage ? [{ url: ogImage }] : undefined,
    },
  };
}
```

每页 `generateMetadata({ params })` 调用 `buildMetadata({...})`。

### 8.2 `app/sitemap.ts`

读取 `features` 与 `getProductStaticParams()`，输出（locale × 启用页面 + locale × 全产品 slug）的 URL 数组。禁用页面不输出。每条 url 含 `lastModified: new Date('2026-06-15')`、`changeFrequency: 'monthly'`、`alternates.languages` 给 hreflang。

### 9.3 自定义事件（`lib/analytics.ts`）

```ts
'use client';
import { track as vercelTrack } from '@vercel/analytics';

export type AnalyticsEvent =
  | 'product_card_click'
  | 'contact_email_click'
  | 'contact_phone_click'
  | 'contact_whatsapp_click'
  | 'search_query'
  | 'language_switch';

export function track(name: AnalyticsEvent, props: Record<string, string | number>) {
  if (process.env.NODE_ENV === 'development') {
    console.debug('[track]', name, props); return;
  }
  vercelTrack(name, props);
}
```

事件埋点表（与 ARCHITECTURE 第 11/12 节一致）：

| 事件 | 触发位置 | 属性 |
|---|---|---|
| `product_card_click` | ProductCard、RelatedProductCard | `slug`, `source` ('products_featured' / 'detail_related' / 'home_categories') |
| `contact_email_click` | TrackedLink (mailto) | `page`, `product_slug` |
| `contact_phone_click` | TrackedLink (tel) | `page` |
| `contact_whatsapp_click` | TrackedLink (wa.me) | `page`, `product_slug` |
| `search_query` | ProductSearchBox onChange (debounced 400ms) | `query`, `results_count` |
| `language_switch` | LanguageSwitcher | `from`, `to`, `page` |

V0.1 不实现 `pdf_download`（无下载），不实现 `unlisted_product_inquiry`（已含在 email_click + page 属性中）。

const files = await glob('src/{content,messages}/**/*.{ts,json,md}');
const hits: { file: string; line: number; text: string }[] = [];
